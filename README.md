# FitCore ERP — Overhauled Landing Page

A premium, production-ready, enterprise-grade SaaS landing page for **FitCore ERP**, built with modern design principles (glassmorphism, vibrant colors, custom animations, and responsiveness).

---

## ✨ Features Implemented

*   **Elite UI/UX & Responsive Layouts**: Fully responsive grids, custom navigation, and device mockup scaling.
*   **Rotating Typing Animation**: Lightweight, vanilla JS word rotator in the hero section.
*   **Staggered Gym Dumbbell Lift Animation**: Slow keyframe floating animations on cards, staggered to simulate lifting reps, which pauses smoothly on mouse hover.
*   **Tabular Numbers & Grid Stabilization**: Eliminated metrics layout "shaking" by using `font-variant-numeric: tabular-nums` and stable min-widths.
*   **Vercel-style Scroll Reveals**: Alternating directional entrance reveals (`left`, `right`, `top`, `bottom`) with unblur transitions.
*   **Swipe/Drag Testimonial Carousel**: Touch-friendly swipe transitions for member reviews.
*   **Interactive Components**:
    *   Smooth Accordion-style FAQ.
    *   Glassmorphic Sign In Modal.
    *   Step-by-step Start Free Trial modal with database provisioning progress loader.
*   **SEO & Web Standards**: Structured metadata, Schema.org JSON-LD data block, semantic elements, and accessibility indicators.

## 🛠️ Built With
*   HTML5
*   CSS3 (Vanilla CSS with HSL variables)
